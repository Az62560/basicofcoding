Apprendre CSS

CSS représente les feuilles de style en cascade. C’était pour créer durant les années 90. C’est dans le but d’ajouter du style aux mots clés HTML. Avec CSS, vous pouvez changer les couleurs, les tailles et éventuellement créer des animations !

h1{

color : #fff;
font-size: 25px;
font-weight: 700;
}

Deux outils sont vraiment importants en CSS : flex et grid.
